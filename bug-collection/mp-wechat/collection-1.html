# 微信小程序富文本问题

# 修改富文本内的样式

data.content = data.content.replace(/\<p/gi, '<p style="color: #fff;"');

# 使用微信原生组件，出现堆栈溢出

<rich-text nodes="{{richText}}"></rich-text>
// 报错
RangeError: Maximum call stack size exceeded

# 解决办法

使用wxParse（微信小程序富文本解析自定义组件，支持HTML及markdown解析）

# 有些标签没有解析出来

# 解决办法（通过正则匹配进行全局替换）

let content = resData.data.content
    .replaceAll("<spanyes';", "<span style=\"")
    .replaceAll("</spanyes([^>]*)>", "</span>")
    .replaceAll("=\"\"", "")
    .replaceAll("<o:p>", "") // 替换完发现有很多<o:p> 直接去掉
WxParse.wxParse('article', 'html', content, this, 5)

TIP

替换完成后发现仍有一些奇奇怪怪的标签，用到的解决办法是，找到对应的地方，删除掉空格或者换行后重新用键盘输入空格或者换行，然后页面上的错误标签就没有了。
