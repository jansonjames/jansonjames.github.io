请求方法中采用了concat或者es6扩展方式拼接数据

方式1：

    storeRankList: this.data.storeRankList.concat(data)

解决办法：

1、点击1次后将点击事件锁定，xx时间后解锁

if(index == this.data.navIndex && this.data.navLocked) {
    return
}
this.data.navLocked = true
setTimeout(() =>{
    this.data.navLocked = false
}, 500)

2、保存请求接口时的页数，数据返回回来之后，看看数组中该页数对应的位置是不是有数据了，有的话就是重复了，替换对应位置的数据或者不往里加。

3、通过匹配看看接口数据有没有和现存数据相同的，有的话就是重复了，不往里加。
