# 微信小程序端

由于微信对用户头像和昵称的获取接口进行了调整，导致这些接口已经没有什么用途，所以抛弃了通过用户接口获取用户信息的方式，只保留了授权获取手机号登录的方式。

# 获取用户头像昵称的调整

小程序登录、用户信息相关接口调整说明 (opens new window)

小程序用户头像昵称获取规则调整公告 (opens new window)

wx.getUserInfo

无法通过wx.getUserInfo与<button open-type="getUserInfo"/>获取用户个人信息（头像、昵称、性别与地区），将直接获取匿名数据（包括userInfo与encryptedData中的用户个人信息）

wx.getUserProfile

通过 wx.getUserProfile 接口获取用户头像将统一返回默认灰色头像，昵称将统一返回 “微信用户”

<template>
  <view class="wrap">
    <u-button type="primary" open-type="getPhoneNumber" @getphonenumber="getPhoneNumber" ripple>授权微信手机号</u-button>
  </view>
</template>

<script>
  export default {
    methods: {
      getPhoneNumber(e) {
        const {
          encryptedData,
          iv,
          code
        } = e.detail
        this.$g_util.request({
          url: '/api/mpweixin/getPhoneNumber',
          data: {
            code
          },
          method: 'POST'
        })
      }
    }
  }
</script>
